Estructura: Árbol de Decisiones

¿Cómo funciona?

El Mini Akinator utiliza un Árbol Binario de Decisiones. Cada nodo representa una pregunta con dos posibles caminos (Sí/No), reduciendo el conjunto de posibles personajes hasta llegar a una hoja (resultado final).

Mapa de Decisiones

[INICIO] -> Deportista? ├── SÍ -> ¿Fútbol? │ ├── SÍ -> ¿Argentino? -> MESSI │ └── NO -> ¿Americano? -> LEBRON JAMES └── NO -> ¿Político? ├── SÍ -> ¿De USA? │ └── SÍ -> ¿Presidente? │ ├── SÍ -> TRUMP │ └── NO -> CHARLIE KIRK └── NO -> No identificado

Base de Conocimientos

Messi: Deportista + Fútbol + Argentino
C. Ronaldo: Deportista + Fútbol + Portugués
Trump: Político + USA + Presidente
Charlie Kirk: Político + USA + No Presidente
